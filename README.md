## ğŸ§ª How to Run

| Language     | File Extension(s) | Compilation (if needed)                  | Execution Command(s)                                     | Notes                                                                                                                               |
| :----------- | :---------------- | :--------------------------------------- | :--------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------- |
| **C** | `.c`              | `gcc your_c_file.c -o your_executable`   | `./your_executable`                                       | Requires GCC compiler to be installed.                                                                                              |
| **C++** | `.cpp`, `.cc`, `.cxx` | `g++ your_cpp_file.cpp -o your_executable` | `./your_executable`                                       | Requires G++ compiler to be installed.                                                                                              |
| **C#** | `.cs`, `.csproj`  | `dotnet build your_project.csproj` (optional) | `dotnet run your_project.csproj` or `./path/to/executable` (after publish) | Requires the .NET SDK to be installed. Execution often involves the project file.                                                    |

| **Python** | `.py`              | None (interpreted)                       | `python your_python_script.py` or `python3 your_script.py` | Requires the Python interpreter to be installed. Use `python` or `python3` depending on your system.                                  |
| **JavaScript** | `.js`              | None (interpreted)                       | `node your_javascript_file.js`                             | Requires Node.js runtime environment to be installed for running outside a browser.                                              |
